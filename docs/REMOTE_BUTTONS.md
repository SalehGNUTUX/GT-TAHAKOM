# أزرار الريموت — التصنيف العام (لكل الأجهزة)

التطبيق **عام ومدفوع بالبيانات**: لا يخصّص لجهاز بعينه. كل جهاز يعرض تلقائياً
الأزرار التي يدعمها فعلاً (من قاعدته أو بروتوكوله الشبكي)، موزّعة بين الشاشة الرئيسية
وقائمة "المزيد" (⋮). مرجع الأزرار مستمدّ من ريموتات واقعية (LG AKB75675325، Philips...).

## الشاشة الرئيسية (الأزرار الأكثر استخداماً)
- **علوي:** الطاقة (POWER) · الرئيسية (HOME) · القائمة (MENU) · رجوع (BACK).
- **لوحة الاتجاهات:** فوق/تحت/يمين/يسار (NAV_*) + OK (NAV_OK) — **أزرار قابلة للنقر**
  (مواضع ثابتة لا تنعكس في RTL).
- **روكرات:** الصوت (VOL_UP/DOWN + MUTE) · القناة (CH_UP/DOWN).
- **الوسائط:** ترجيع/تشغيل/إيقاف مؤقت/تقديم (RWD/PLAY/PAUSE/FFWD).
- **الترويسة:** مصدر (SOURCE) + زر المزيد (⋮).

## قائمة "المزيد" (⋮) — أزرار إضافية
- **الأرقام:** 0–9 (DIGIT_*) — لوحة شبكية.
- **الوظائف:** دليل البرامج (GUIDE) · قائمة القنوات (LIST) · معلومات (INFO) ·
  الترجمة (CC) · القائمة الذكية/التطبيقات (SMART) · الإعدادات (SETTINGS) ·
  المصدر (SOURCE) · خروج (EXIT) · تيليتكست (TEXT).
- **الأزرار الملوّنة:** أحمر/أخضر/أصفر/أزرق (RED/GREEN/YELLOW/BLUE) — للتيليتكست والتفاعل.

## مرجع ريموت LG (AKB75675325) — الأزرار الكاملة
POWER · SUBTITLE(CC) · SETTINGS · SOURCE/INPUT · أرقام 0-9 · LIST · ⋯(المزيد) ·
AD(Quick Access) · GUIDE · VOL± · MUTE · CH± · NETFLIX/PRIME(اختصارات تطبيقات) ·
HOME · BACK · EXIT · الاتجاهات+OK · TEXT · T.OPT · MOVIES · REC · STOP ·
RWD/PLAY/PAUSE/FFWD · الأزرار الملوّنة الأربعة.

## ملاحظات للتنفيذ المستقبلي
- **اختصارات التطبيقات** (Netflix/Prime/YouTube): تحتاج أكواد IR خاصة (موجودة في بعض
  ريموتات LG) أو أوامر شبكية (webOS/Android TV). تُضاف عند توسيع القاعدة/البروتوكولات.
- **التلفاز الذكي عبر الشبكة** (LG webOS, Android TV): التنقّل داخل التطبيقات (يوتيوب…)
  وتشغيل/إيقاف المقاطع يتطلّب بروتوكول الشبكة (SSAP/Remote v2) لا الـ IR — قادم في
  مراحل بروتوكولات الشبكة. الـ IR يغطّي التحكّم الأساسي فقط.
- أي زر جديد يُضاف إلى `ButtonId` ويُربط في كل Transport (IR عبر القاعدة، Roku/شبكة عبر الخريطة).
