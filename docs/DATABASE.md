# قاعدة بيانات الأجهزة والأكواد — محلي أوفلاين أم يتطلّب إنترنت؟

> العربية · [English](en/DATABASE.md)

**المبدأ المعتمد: أوفلاين افتراضياً.** كل الوظائف الأساسية تعمل بلا إنترنت.
الإنترنت اختياري فقط لتوسيع أكواد IR لعلامات نادرة، وما يُنزَّل يُحفظ محلياً فلا يُطلب ثانية.

التصميم طبقتان منفصلتان تماماً:

## 1. اكتشاف الأجهزة الشبكية (م1) — لا قاعدة بيانات، أوفلاين 100%

التلفازات الذكية وصناديق أندرويد (أصلية وصينية) وأجهزة IP **تُعلن عن نفسها** على شبكة
WiFi المحلية. التطبيق يلتقط الإعلان (اسم، نوع، أحياناً طراز) ويستنتج البروتوكول عبر
`ServiceFingerprint` — **دون إنترنت ودون أي قاعدة مخزّنة**. الجهاز نفسه هو "القاعدة".

## 2. أكواد IR (م4) — هنا تظهر القاعدة، بتصميم هجين أولويته محلي

| الطبقة | المصدر | إنترنت؟ | الحالة |
| :-- | :-- | :-- | :-- |
| **أساسية مدمجة** | `assets/irdb/` — 60 جهازاً (TV 34 + Cable 19 + Audio 7) من IRRemote + استيراد NEC/RC5/RC6 من probonopd/irdb، بصيغة موحّدة (index.json + ملف/جهاز، Pronto + تردد). | ❌ أوفلاين كلياً | ✅ منجز |
| **توسعية** | البحث الشبكي في probonopd/irdb: فهرس 3244 طقماً مشحون كأصل (بحث أوفلاين) + تنزيل أكواد الطقم المختار عبر CDN وتحويله Pronto على الهاتف (`IrCodeConverter`) وحفظه محلياً. | ✅ الجلب فقط أونلاين | ✅ منجز (0.9.9) |
| **حزم المستخدم** | ملفات `.tahakom` المشتركة (انظر [SHARING.md](SHARING.md)). | ❌ أوفلاين | منجز جزئياً |
| **التعلّم (Learn)** | التقاط الإشارة من ريموت أصلي عبر مستقبِل الهاتف (إن وُجد). | ❌ أوفلاين | مخطّط (م4) |

### لماذا أولوية المحلي؟
- الاستخدام منزلي: قد يكون الهاتف على WiFi محلية بلا إنترنت فعّال.
- الخصوصية والسرعة: لا اعتماد على خادم خارجي للوظائف الأساسية.
- المتانة: التطبيق لا "يتعطّل" إذا غاب الإنترنت أو أُغلقت قاعدة LIRC مستقبلاً.

### التخزين التقني
- **Room**: الأجهزة المحفوظة + الريموتات المخصّصة + أكواد المستخدم + كاش ما نُزِّل.
- **assets/**: القاعدة المدمجة للقراءة فقط (تُحمَّل عند أول طلب علامة).
- **DataStore**: التفضيلات (اللغة، آخر شبكة، خيار "السماح بالتنزيل أونلاين").

## الخلاصة

| الوظيفة | تحتاج إنترنت؟ |
| :-- | :-- |
| اكتشاف الأجهزة الشبكية (م1) | ❌ لا |
| التحكّم عبر الشبكة (م2–م3) | ❌ لا (شبكة محلية) |
| أكواد IR لأشهر العلامات (م4) | ❌ لا (مدمجة) |
| أكواد IR لعلامة نادرة غير مضمّنة | ✅ مرة واحدة فقط (ثم تُحفظ) |
| استيراد/مشاركة حزم `.tahakom` | ❌ لا |
